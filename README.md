# CJAS PYTHON E DADOS - DATA DO EVENTO: 9/10-12/10      

Repositório criado para desenvolvimento do sistema de ánalise dos dados das incrições para o Acampamento do JAS do ano de 2026. Esse repositório traz anotações sobre os treinamentos, cronograma, códigos do google collab (usado para análise e tratamento de dados) e todos os códigos e requirements necessários (apenas é preciso baixar as bibliotecas e extensões).

# Ferramentas utilizadas:

- Python - Bibliotecas: Matplotlib, Pandas, Seaborn 
- VsCode/Google Collab
